## Функционал

Планирую написать сайт интернет магазина книг на Django, на котором пользователи:
- cмогут регистрироваться
- выбирать книги по категориями из каталога
- смогут добавлять товары в корзину и оформлять заказ
- смогут обратиться к система геолокации для проверки наличия определенной книги в конкретном городе
- смогут использовать телеграм-бот, который будет отвечать на вопросы клиентов  и уведомлять о поступлении книги

## Стек технологий
