Функционал

Планирую написать сайт интернет магазина книг на Django, на котором пользователи могут регистрироваться, выбирать книги по категориями из каталога. Пользователи смогут добавлять товары в корзину и оформлять заказ. На сайте  будем система геолокации для проверки наличия определенной книги в конкретном городе, а также телеграм-бот, который будет отвечать на вопросы клиентов  и уведомлять о поступлении книги.

Стэк технологий 

- Python (Django)
- JS,HTML,CSS
